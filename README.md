# Agent Skills

A collection of agent skills that extend capabilities across planning, development, and tooling.

## Planning & Design

These skills help you think through problems before writing code.

- **brainstorming** — Transforms vague ideas into validated designs through structured dialogue and disciplined reasoning before any implementation begins.

  ```
  npx skills@latest add brenont/skills/brainstorming
  ```

- **design-an-interface** — Generates multiple radically different interface designs for a module using parallel sub-agents, then compares and synthesizes the best approach.

  ```
  npx skills@latest add brenont/skills/design-an-interface
  ```

- **grill-me** — Interviews you relentlessly about a plan or design, walking down each branch of the decision tree until reaching shared understanding.

  ```
  npx skills@latest add brenont/skills/grill-me
  ```

## Development

These skills help you write, refactor, and fix code.

## Tooling & Setup

- **write-a-skill** — Creates new agent skills with proper structure, progressive disclosure, and bundled resources.

  ```
  npx skills@latest add brenont/skills/write-a-skill
  ```
